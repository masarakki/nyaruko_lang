The nigh indescribable programming language Nyaruko
====

I'm the chaos that always creeps up to your side with a friendly smile, Nyarlothotep language!

Spec
----

It's a language in the same category as Brainfuck

- *(」・ω・)」うー(／・ω・)／にゃー* - Move the pointer to the right
- *(」・ω・)」うー!(／・ω・)／にゃー!* - Increment the value the pointer is pointing to by 1
- *(」・ω・)」うー!!(／・ω・)／にゃー!!* - Move the pointer to the left
- *(」・ω・)」うー!!!(／・ω・)／にゃー!!!* - Decrement the value the pointer is pointing to by 1
- *CHAOS☆CHAOS!* - If the value the pointer is pointing to is 0 then jump up to the corresponding "I WANNA CHAOS!"
- *I WANNA CHAOS!* - If the value the pointer is pointing to is not 0 then jump up to the corresponding "CHAOS☆CHAOS!"
- *Let's＼(・ω・)／にゃー* - Output the value the pointer is pointing to
- *cosmic!* - Read 1 byte from input

All characters besides those and newlines will be ignored.
